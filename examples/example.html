• Example: basic

    Demonstrates the most basic usage.

  • Example: cancelling

    Demonstrates that you can cancel a callback which will cause it not to run.

    When the block stops spinning nothing should be shown in the log.

  • Example: copied object

    Demonstrates that you can create new TransitionCallback objects and that copied objects gets a new list of callbacks.

    'Transition finished' should only be printed once.

  • Example: timeout

    Demonstrates a callback that will be called if the object hasn't finished animating in 100ms.

    When the callback has timed out the transition is started for real. Nothing should be printed when the transition is done.

  • Example: duplicates

    This will show that creating another callback with the same ID will cancel the first one.

  • Example: jQuery

    This will show how to use this library with jQuery.
